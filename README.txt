MATLAB code for "Ocular Artifacts Elimination from Multivariate EEG Signal using Frequency-Spatial Filtering"

If you are using this code please cite the following article:

A. Bhattacharyya, A. Verma, R. Ranta and R. B. Pachori, "Ocular Artifacts Elimination From Multivariate EEG Signal Using Frequency-Spatial Filtering," in IEEE Transactions on Cognitive and Developmental Systems, vol. 15, no. 3, pp. 1547-1559, Sept. 2023, doi: 10.1109/TCDS.2022.3226775.

Main file: EWT_SCA_OA_Removal.m

Sample EEG Signal: SampleEEGSig.mat (Available in https://physionet.org/content/eegmmidb/1.0.0/  , G. Schalk, D. J. McFarland, T. Hinterberger, N. Birbaumer, and J. R.
Wolpaw, “Bci2000: a general-purpose brain-computer interface (bci) system,” IEEE Transactions on Biomedical Engineering, vol. 51, no. 6, pp. 1034–1043, 2004.)

This MATLAB code uses Empirical Wavelet Transform (EWT) Toolbox by Jerome Gilles (https://in.mathworks.com/matlabcentral/fileexchange/42141-empirical-wavelet-transforms)
